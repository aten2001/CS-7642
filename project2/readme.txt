Code Location: https://github.com/thatguyandy27/CS-7641/tree/master/project2

Run this code requirements

1. Python3
2. JYTHON
3. anaconda libraries
    - scipy
    - pandas
    - numpy
    - matplotlib
4. Put the data in the ./data/ folder
    - white wine dataset
        - Download from https://archive.ics.uci.edu/ml/datasets/Wine+Quality
        - Put the winequality-white.csv file in the ./data/wine/ folder.
    - diabetes dataset
        - Download from https://archive.ics.uci.edu/ml/datasets/Diabetic+Retinopathy+Debrecen+Data+Set
        - Remove the first 24 lines from the file (until @data)
        - Put the data.text file in the ./data/diabetes_renop/ folder.
4. Run 'python run_experiment.py --dump_data'
5. Run 'jython NN-Backprop.py'
6. Run 'jython NN-GA.py'
7. Run 'jython NN-RHC.py'
8. Run 'jython NN-SA.py'
9. Run 'jython continuouspeaks.py'
10. Run 'jython flipflop.py'
11. Run 'jython tsp.py'
12. Run 'python plotting.py'
13. Results are sent to the ./output/ folder
14. Images are sent to the ./output/images/
